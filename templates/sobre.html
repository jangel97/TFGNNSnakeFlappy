Sobre Nosotros

Porqué hacemos esto.

En este trabajo vamos a ver cómo funcionan las redes neuronales. De tal manera que se verá de donde sale la idea de red neuronal artificial, que partes la constituyen, como funciona cada una de estas y qué algoritmos se requieren para su funcionamiento. También investigaremos algunas de las diferentes librerías con las que podemos crear una red neuronal y discutiremos cual da mejores resultados.

Además, en la parte práctica veremos la estructura y implementación de una arquitectura cliente-servidor para que varios juegos se puedan comunicar con la red neuronal y que esta tome decisiones para jugar a cada uno de ellos. Es por esto que se ha implementado un sistema genérico.

El sistema consta de un juego, como cliente, que se comunicará con el servidor, a través de un api rest, que contendrá una red neuronal la cual estará entrenada para dicho juego o tendrá que ser entrenada. Esta red neuronal predecirá qué movimiento se tiene que hacer en el juego para así hacer el máximo de puntos posibles.

Las motivaciones para este trabajo no son otras que aprender cómo funciona un red neuronal, ver las características de esta y demostrar lo aprendido durante el grado de informática con una gran capacidad de aprendizaje rápido y adaptación sobre una materia que ninguno de los dos alumnos hemos trabajado nunca antes, las redes neuronales.
